# Research Topic

Research a topic and compile findings into a brief.

## When to use
- Triggered by "research", "look into", "what's the latest on"
- When the user needs background on a subject

## Steps

1. Search for recent information on the topic
2. Check multiple sources for accuracy
3. Compile findings into a structured brief:
   - Summary (2-3 sentences)
   - Key findings (bullet points)
   - Sources (with links where available)
   - Recommendations or next steps
4. Save the brief to memory/ for future reference
5. Send summary to user via messaging channel

## Format

Keep the summary under 500 words. Lead with the most important finding. Cite sources.
